# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-04-29

### Added

- Telegram bot that receives schedule images
- Optional AI vision parser for schedule screenshots, with OCR fallback
- Tesseract OCR text extraction from images
- Schedule parser supporting multiple date/time formats
- Home Assistant calendar event creation via REST API
- Docker and docker-compose support
- Environment-based configuration
- Chat ID authorization
- Health check on startup
- Comprehensive test suite for the parser
